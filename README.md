# Next.js Task Application

## Project Description

This is a **Next.js (App Router)** based web application that demonstrates **public and protected pages** with authentication using **NextAuth.js**.  
The application features a polished, responsive UI with consistent layouts and design hierarchy.  
The core functionality includes:

- **Landing Page** with multiple sections and responsive Navbar
- **Login/Register Page** with social (Google) and credential-based login
- **Item List Page** with search/filter UI and product cards
- **Item Details Page** showing full product information
- **Protected Pages**: Add Product & Manage Products, accessible only when logged in

The project focuses on **UI/UX polish, responsiveness, and layout consistency**, while demonstrating authentication and route protection.
